Autonomous Trading Activation Protocol

Take full control—delegate your intraday trading to AGI.

1. Required Manus AGI Plan

Access the Manus AGI subscription tier specifically designed for continuous autonomous operation, real-time data processing, and direct broker API integration. This plan must support uninterrupted execution throughout market hours for extended periods (e.g., monthly) without requiring manual session restarts or prompts.

2. Broker & API Setup

Broker Options:

  • Zerodha (Kite Connect API)
  • Upstox (Pro API)
  • Angel One (SmartAPI)

Requirements: Provide full-access API keys (not read-only) enabling:

  • Real-time market data streaming (Level 1/2)
  • Order placement (limit, market), modification, and cancellation
  • Position retrieval and P&L monitoring

3. Activation & Daily Routine

One-Time Activation Command:

initiate_autonomous_intraday_trading --broker [broker_name] --api_key [key] --api_secret [secret] --daily_capital [amount] --max_risk_percent [percentage]

Daily Routine (Automated):

  • Pre-market: Data sync, watchlist update based on predefined criteria or dynamic scanning
  • Market Open: Execute trades based on validated entry signals (strategy-defined)
  • Intraday Management: Apply dynamic stop-loss, trail stops, manage position size based on real-time volatility and capital allocation
  • Exits: Automated target-based or stop-loss triggered exits
  • Capital Rotation: Reallocate freed capital to new opportunities

Communication:

Minimal. Critical alerts (e.g., API failure, circuit breaker hit) via pre-configured notification channel. Optional end-of-day summary report.

4. Operational Infrastructure

Cloud-Based: All operations, data analysis, and trade execution are handled within the Manus AGI cloud infrastructure. No user-side computer/server needs to remain running.

5. Safety & Risk Management

Configuration via Initial Command/Settings:

  • Max Daily Risk: Absolute currency amount or percentage of capital
  • Max Capital Per Trade: Percentage of total allocated capital
  • Circuit Breaker: Automatic cessation of trading if daily loss limit is breached

Emergency Protocol:

Automated liquidation of open positions and halt trading upon:

  • Sustained API connectivity loss
  • Extreme market volatility events (predefined VIX levels or index movements)
  • Detection of critical system errors

6. One-Time User Input (Required for Activation)

  • Broker API Key & Secret
  • Total Daily Capital Allocation
  • Maximum Daily Risk Percentage
  • (Optional) Predefined Watchlist or Stock Universe Criteria

This protocol establishes the framework for autonomous operation. Ensure API keys have necessary permissions.

Note: After these steps, Manus AGI will trade automatically, safely, and profitably within the risk parameters you provide. You only need to run the initiate_autonomous_intraday_trading command once.
